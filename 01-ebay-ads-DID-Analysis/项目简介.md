# 基于双重差分法（DiD）的 eBay 搜索广告 ROI 深度评估
![EBay_logo](https://github.com/user-attachments/assets/ce4b9840-eb3a-4eb9-b067-6d945f7def1d)
    
## 1. 项目概述
### 1.1 业务背景与核心问题
在搜索引擎营销（SEM）中，企业常面临一个经典的归因难题：**付费搜索广告究竟带来了真实的增量销售，还是仅仅蚕食了原本就会发生的自然搜索流量（替代效应）？**
为了精确评估搜索广告的真实投资回报率（ROI），eBay 在全美选定的指定市场区域（DMA）进行了一项 A/B 实验。在实验组 DMA 全面暂停搜索广告投放，同时保留对照组 DMA 的正常广告活动。本项目旨在通过高阶因果推断模型，剥离自然市场波动，量化广告暂停对核心商业指标的真实冲击。

### 1.2 实验设计
**实验设计：** 将美国市场战略性地划分为 210 个互不重叠的指定市场区域（DMA），作为独立的实验单元，以减轻溢出效应。
**实验组与对照组分配：**
* **实验组（干预组）：** 随机抽取一部分 DMA，在数月的观察期内，全面暂停所有非品牌“通用”关键词（例如“跑步鞋”）的付费搜索广告。
* **对照组（照常营业组）：** 剩余的 DMA 维持其历史基准的搜索广告支出和竞价策略。

**分析目标：** 严格剥离并量化通用搜索广告对总销售额的真实“因果提升”（增量价值）。这种地理划分有效地控制了不可观测的地区异质性、季节性趋势和宏观经济冲击。

## 2. 数据概览与预处理
数据集包含 18,200 行数据，每一行代表 A/B 测试期间某个特定日期的一个指定市场区域（DMA）。核心变量包括：

| 变量 | 描述 |
| :--- | :--- |
| **DMA** | 指定市场区域（DMA）的名称，是用于广告定向和分析的地理区域。 |
| **Code** | 分配给每个 DMA 的唯一数字标识符（例如，纽约为 501）。 |
| **Date** | 观测的日历日期，涵盖实验持续时间（2024 年第二季度）。 |
| **Population** | DMA 的估计人口。该变量有助于在分析中控制市场规模。 |
| **MedianHHIncome** | DMA 的家庭收入中位数。捕捉可能影响消费者支出行为的经济差异。 |
| **Broadband** | DMA 内接入高速互联网的家庭百分比。作为数字基础设施和在线购物便利性的代理指标。 |
| **Urbanization** | 代表 DMA 城市化程度的分数（数值越高表示城市化程度越高）。影响搜索行为、设备使用和购物模式。 |
| **eCommIndex** | 内部开发的指数，表明 DMA 居民参与电子商务的整体倾向。数值越高反映市场数字化参与度越高。 |
| **MedianAge** | DMA 居民的年龄中位数。可用于控制在线购物偏好中的人口统计学差异。 |
| **CollegeDeg** | DMA 中拥有大学及以上学历的成年人百分比。可能与数字素养或品牌认知度相关。 |
| **CostIndex** | DMA 的综合生活成本指数，按比例缩放，100 代表全国平均水平。数值越高表示该地区生活成本越高。 |
| **Treatment** | 二元指示变量（0 或 1），表示 DMA 是否被分配到实验组（在 Post 期间关闭广告）。 |
| **Post** | 实验阶段的二元指示变量（0 或 1）。0 表示干预前（Pre-treatment）阶段（所有 DMA 均投放广告），1 表示干预后（Post-treatment）阶段（实验组 DMA 停止投放广告）。 |
| **DayIndex** | 连续的日期计数器，从 2024 年 4 月 1 日的 0 开始。可用于时间序列绘图和趋势分析。 |
| **AdSpend** | eBay 当天在该 DMA 的付费搜索广告上的支出金额。对于处于 Post 期间的实验组 DMA，此值为 0。 |
| **PaidClicks** | 当天该 DMA 用户点击 eBay 付费搜索广告的次数。 |
| **OrganicClicks** | 来自自然（未付费）搜索结果的点击次数，捕获未消耗广告支出的流量。 |
| **TotalClicks** | 由搜索驱动的总点击次数，计算方式为 PaidClicks 和 OrganicClicks 之和。 |
| **SalesAds** | 归因于通过付费搜索广告到达的用户的销售额（美元）。这有助于估算广告驱动流量的直接回报。 |
| **TotalSales** | 当天该 DMA 内的所有总销售额（美元），包括自然流量和付费流量，以及非搜索来源的销售额。 |

## 3. 分析框架与方法论
为了得出最严谨的商业结论，本项目摒弃了简单的均值对比，部署了一套综合的计量经济学因果推断框架：
* **核心模型演进：** 从将 `Population` 作为异方差缩放因子的基础双重差分模型（DID）开始，分析最终升级为**双向固定效应（TWFE）模型**。
* **噪音粉碎机制：** 通过引入 DMA 固定效应，模型完美吸收了不同地区长期的经济与文化基线差异。时间固定效应的加入有效剥离了影响全网的宏观层面波动和星期几的季节性效应。
* **交叉验证：** 额外引入了**合成控制法（SCM）**，利用机器学习优化的权重从供体池中“克隆”出一个合成的实验组，从而对 TWFE 模型进行了史诗级的交叉验证。

### 3. 核心指标发现
基于实验前对照组的日均基准水平，模型得出以下精确的因果效应估计：

**A. 政策执行确认：广告支出与付费点击**
* **AdSpend（广告支出）**: $\beta_3 = -44.14$ ($p < 0.001$)
* **PaidClicks（付费点击）**: $\beta_3 = -44.11$ ($p < 0.001$)
* **结论：** 实验执行严格。实验组的日均广告支出显著下降了约 44 美元，相应的付费点击量也减少了约 44 次。

**B. 替代效应测试：自然流量是否弥补了损失？**
* **OrganicClicks（自然点击）**: $\beta_3 = +1.97$ ($p = 0.827$)
* **结论：** 自然点击量的增加在统计学上极不显著。这表明用户并没有仅仅因为广告消失就大规模转向点击自然链接。**替代效应极弱。**

**C. 整体流量与销售额的真实冲击**
* **TotalClicks（总点击）**: $\beta_3 = -42.14$ ($p < 0.001$)。总流量显著下降，流失的流量规模大致相当于付费点击的减少量。
* **TotalSales（总销售额）**: $\beta_3 = -17.54$ ($p = 0.001$)。在日均约 2,806 美元的基准下，暂停广告导致实验组的日均总销售额出现了 **17.54 美元**的纯粹因果下降。

---

### 4. 鲁棒性检验：验证因果推断
为了确保 17.54 美元的日销售额下降是绝对的因果效应而非统计伪影，本项目进行了两项极其严苛的计量经济学验证测试：

**A. 平行趋势假设检验 (Event Study / Dynamic DID)**

通过将时间轴重构为相对周数（以 $T=-1$ 为基准），动态 DID 模型表明，在干预前（$T < 0$），所有估计系数的 95% 置信区间完美穿过 0 轴，证实了实验前两组具有统计学上完全一致的销售轨迹。在政策实施后（$T \ge 0$），趋势线瞬间暴跌并持续保持在 0 轴下方。这完美地验证了平行趋势假设，并证明负面冲击是即时且持久的。

**B. 空间安慰剂检验 (Permutation Test)**

为了排除随机市场波动和遗漏变量的干扰，执行了包含 200 次迭代的蒙特卡洛模拟。通过随机分配“伪实验组”身份并重新运行 TWFE 模型，由此产生的 200 个“伪因果效应”形成了一个完美的以零为中心的正态分布。而实际得出的极端效应（-17.54 美元）远远落在了这种随机分布的左侧长尾之外（$p\text{-value} \approx 0.000$）。这无懈可击地证明了观察到的销售额损失并非巧合。

---

### 5. 商业洞察与战略建议

综合上述经过多重验证的数据，我们可以对这些搜索广告的商业价值做出最终定性评估：

1. **这些广告具有“增量价值”吗？—— 是的。**
   数据彻底打破了内部“自然流量会完全替代付费流量”的假设。这些广告确实为 eBay 带来了大量的额外流量和真实的销售额，而不是仅仅作为毫无意义的重复曝光。
2. **这些广告“盈利”吗？—— 严重亏损。**
   计算其真实的广告支出回报率（ROAS）：
   * 节省的日均广告成本：**$44.14**
   * 损失的日均总销售额：**$17.54**
   * **真实 ROAS** = 17.54 / 44.14 **$\approx 0.40$**

**【最终战略建议】**
针对本次实验涵盖的特定搜索广告（可能是缺乏新客获取能力或用于老客召回的品牌关键词广告），虽然它们确实产生了一定的真实增量，但其**获客成本高得离谱**（每投入 1 美元仅产生 0.40 美元的营业额）。 
**强烈建议 eBay 永久暂停这些特定的搜索广告**，或实施极为激进的降价竞价策略。释放出的预算池（每个 DMA 每天 44 美元）应全额重新分配给内部其他已知 ROAS > 1 的高收益营销渠道，以实现全局利润的最大化。
