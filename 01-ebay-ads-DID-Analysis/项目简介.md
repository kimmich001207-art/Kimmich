# 项目展示：基于因果推断框架的 eBay 搜索广告 ROI 深度评估

## 1. 业务背景与核心问题 (Business Context & Core Questions)
在搜索引擎营销（SEM）中，企业常面临一个经典的归因难题：付费搜索广告究竟带来了真实的增量销售，还是仅仅蚕食了原本就会发生的自然搜索流量（替代效应）？

为了精确评估搜索广告的真实投资回报率（ROI），eBay 在全美选定的城市区域（DMA）进行了一项 A/B 实验：在实验组城市全面暂停搜索广告投放，同时保留对照组城市的正常投放。本项目旨在通过高阶因果推断模型，剥离自然市场波动，量化广告暂停对核心商业指标的真实冲击。

## 2. 分析框架与因果推断方法 (Analytical Framework & Methodology)
为了得出最严谨的商业结论，本项目摒弃了简单的均值对比，采用了一整套先进的计量经济学因果推断框架：

* **核心模型演进：** 从加入了 Population（人口规模）作为异方差缩放因子的基础双重差分模型（DID），最终升级为双向固定效应模型（Two-Way Fixed Effects, TWFE）。
* **噪音粉碎机制：** * 通过引入“城市个体固定效应（DMA Fixed Effects）”完美吸收了不同地区长期的经济与文化基线差异；
    * 通过引入“时间固定效应（Date Fixed Effects）”有效剥离了全网周末效应及宏观波动。
* **交叉验证：** 额外引入基于机器学习优化权重的合成控制法（Synthetic Control Method, SCM），利用对照组池“克隆”出合成实验组，与 TWFE 模型进行史诗级交叉验证。

## 3. 核心指标发现 (Key Metric Findings)
基于控制组实验前的日均基线水平，模型得出了以下精确的因果效应估计：

### A. 政策执行确认：广告支出与付费点击
* **AdSpend（广告支出）：** $\beta_3 = -44.14$ ($p < 0.001$)
* **PaidClicks（付费点击）：** $\beta_3 = -44.11$ ($p < 0.001$)
* **结论：** 实验执行严格，实验组的广告日均花费显著下降了约 44 美元，付费点击量同步减少约 44 次。

### B. 替代效应测试：自然流量是否弥补了损失？
* **OrganicClicks（自然点击）：** $\beta_3 = +1.97$ ($p = 0.827$)
* **结论：** 自然点击量的上升在统计学上极不显著。这意味着用户并没有因为广告消失而大规模转向点击下方的自然链接，替代效应极弱。

### C. 整体流量与销售额的真实冲击
* **TotalClicks（总点击）：** $\beta_3 = -42.14$ ($p < 0.001$)。总流量显著下降，流失量基本与付费点击的减少量持平。
* **TotalSales（总销售额）：** $\beta_3 = -17.54$ ($p = 0.001$)。在基线日均销售额约 2806 美元的背景下，暂停广告导致实验组的日均总销售额产生了 17.54 美元的纯粹因果下降。

## 4. 鲁棒性检验：因果关系的极度确信 (Robustness Checks)
为了确保 17.54 美元的销售额下降是绝对的因果效应，而非统计伪影，本项目进行了两项极其严苛的检验：

### A. 平行趋势检验 (Event Study / Dynamic DID)
通过将时间轴重组为相对周数（以 $T=-1$ 周为基准），动态 DID 模型显示：在干预前（$T < 0$），所有估计系数的 95% 置信区间均完美穿过 0 轴，证实了两组在实验前具有完全平行的销售趋势。在跨过干预点（$T \ge 0$）后，趋势线瞬间跌破 0 轴并持续位于负值区间。这完美验证了平行趋势假设，证明负面冲击是即时且持久的。

### B. 空间安慰剂检验 (Spatial Placebo Test)
为了排除随机波动和遗漏变量的干扰，利用蒙特卡洛模拟执行了 200 次“随机抓阄”证伪实验。通过随机分配“假实验组”身份并重新运行 TWFE 模型，得到的 200 个“假因果效应”呈现出完美的零均值正态分布。而真实算出的极端效应（-17.54 美元）远远落在了该随机分布的左侧长尾之外（$P\text{-value} \approx 0.000$）。这无懈可击地证明了结论并非巧合。

## 5. 商业洞察与战略建议 (Business Insights & Strategic Recommendations)
结合上述经过多重验证的数据，我们对这批搜索广告的商业价值进行最终定性：

* **这批广告具有“增量价值”（Incremental Value）吗？—— 是的。**
    数据彻底打破了内部关于“自然流量会完全替代付费流量”的假设。这批广告确实为 eBay 带来了实质性的额外流量和真实销售额，而非毫无意义的重复曝光。
* **这批广告“盈利”（Profitable）吗？—— 严重亏损。**
    计算其真实的广告支出回报率（ROAS）：
    * 节省的日均广告成本：\$44.14
    * 损失的日均总销售额：\$17.54
    * 真实 ROAS = 17.54 / 44.14 $\approx 0.40$

### 【战略最终落地建议】
针对实验所覆盖的这部分搜索广告（推测为缺乏拉新转化率的品牌词广告或老客召回广告），尽管它们能够带来一定的真实增量，但其获客成本极度高昂（每花 1 美元仅带来 0.4 美元营业额）。

建议 eBay 长期暂停该特定渠道的搜索广告投放，或采用更为激进的降价竞价策略，将释放出的预算池（每 DMA 每日 44 美元）全额转移并重新分配至内部其他已知 ROAS > 1 的高回报营销渠道中，以实现全局利润的最大化。
